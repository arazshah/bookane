# مراحل نصب

برای نصب و راه‌اندازی سیستم مستندات، مراحل زیر را دنبال کنید:

## 1. نصب وابستگی‌ها

ابتدا وابستگی‌های پروژه را نصب کنید:

``` bash
pip install -r requirements.txt
```

## 2. اجرای سرور

سرور را با دستور زیر اجرا کنید:

``` bash
python app.py
```

## 3. دسترسی به مستندات

مرورگر خود را باز کنید و به آدرس زیر بروید:

```
http://localhost:5000
```
